about

RailShooter 9000 is a JavaScript/HTML5 browser game which makes extensive use of the CreateJS library (specifically its EaselJS component). It interacts with the DOM via jQuery and handles keyboard inputs with the keymaster library. I built RailShooter 9000 as a student of the May-August 2015 cohort at App Academy in New York, both for practice and out of a longtime enthusiasm for the rail-shooter/bullet-hell/shmup genre.
